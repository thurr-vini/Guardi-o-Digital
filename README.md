# 🛡️ Guardião Digital - Educação em Segurança Cibernética

O **Guardião Digital** é uma plataforma educacional interativa desenvolvida como um projeto de extensão do curso de Ciência da Computação. O objetivo é capacitar usuários comuns a identificarem ameaças online e adotarem práticas seguras no ambiente digital, cobrindo temas como LGPD, Phishing e Segurança em Redes.

## 🚀 Tecnologias e Frameworks

* **Tailwind CSS:** Utilizado para uma interface moderna, responsiva e otimizada via utilitários.
* **JavaScript (ES6+):** Implementação de toda a lógica interativa (Quiz, Modais, Carrossel e State Management do Tema).
* **HTML5 & CSS3:** Estrutura semântica e animações de gradiente personalizadas (@keyframes).
* **Google Fonts & Lucide Icons:** Foco em tipografia legível (Inter) e interface iconográfica intuitiva.

## 🧠 Diferenciais Técnicos e Funcionalidades

Este projeto destaca competências avançadas de desenvolvimento Front-end:

1.  **Lógica Interativa de Quiz:**
    * Sistema de avaliação com feedback em tempo real.
    * Cálculo de pontuação final e mensagens personalizadas baseadas no desempenho do usuário.

2.  **Verificador de Força de Senha (Real-time):**
    * Algoritmo que analisa complexidade (comprimento, caracteres especiais, números e cases) enquanto o usuário digita, fornecendo feedback visual instantâneo.

3.  **UI/UX Avançada com Tailwind:**
    * **Dark Mode Nativo:** Implementado com persistência no `localStorage` e detecção de preferência do sistema operacional.
    * **Carrossel Responsivo:** Slider de conteúdo dinâmico no Hero com navegação por indicadores e transições suaves.
    * **Glassmorphism & Sticky Header:** Efeitos de desfoque de fundo e navegação fixa para melhor usabilidade.

4.  **Acessibilidade e Semântica:**
    * Uso de Smooth Scroll para navegação entre seções.
    * Estrutura de modais dinâmicos para carregamento de conteúdo educacional sem recarregar a página.
